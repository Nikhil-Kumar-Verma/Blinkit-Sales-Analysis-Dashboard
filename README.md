# Blinkit Sales Analysis Dashboard | Power BI
### Dashboard Link: https://app.powerbi.com/links/ysQgDn-OCd?ctid=59196d81-9bb3-47ad-a070-5825c18dbc3c&pbi_source=linkShare
## Project Overview

This Power BI dashboard provides a comprehensive analysis of Blinkit's sales performance, product categories, and outlet operations. The dashboard enables stakeholders to monitor key business metrics, identify sales trends, evaluate outlet performance, and gain insights into customer purchasing patterns.

## Dashboard Objectives

- Analyze overall sales performance.
- Monitor product category contributions.
- Evaluate outlet performance across locations and sizes.
- Compare sales based on item fat content.
- Identify top-performing products and outlet types.
- Support data-driven business decisions.

## Dashboard Pages

### 1. Sales Overview Dashboard

Provides a high-level overview of business performance.

#### Insights:
- Total sales generated across all outlets.
- Sales distribution by item fat content.
- Sales contribution by product category.
- Sales performance across outlet location tiers.
- Interactive filters for outlet size and location type.

### Key Visuals:
- KPI Cards
- Donut Chart
- Bar Charts
- Slicers

---

### 2. Product Analysis Dashboard

Provides detailed insights into product performance.

#### Insights:
- Top-selling products by Item Identifier.
- Sales comparison by Item Type and Fat Content.
- Product-level performance metrics:
  - Total Sales
  - Average MRP
  - Average Visibility
  - Average Weight
- Decomposition Tree for drill-down analysis.

### Key Visuals:
- Clustered Bar Chart
- Stacked Column Chart
- Matrix Table
- Decomposition Tree

---

### 3. Outlet Analysis Dashboard

Analyzes outlet-level performance.

#### Insights:
- Sales by Outlet Type.
- Sales by Outlet Size.
- Sales by Establishment Year.
- Sales distribution across outlet location tiers.
- Outlet-wise performance comparison.

### Key Visuals:
- Bar Charts
- Donut Chart
- Matrix Table
- Trend Analysis Chart

## Business Insights

### Product Performance
- Fruits & Vegetables generated the highest sales.
- Snack Foods and Household products were among the top contributors.
- Low Fat products contributed a larger share of sales compared to Regular products.

### Outlet Performance
- Supermarket Type1 generated the highest revenue.
- Tier 3 locations recorded the highest sales.
- Medium-sized outlets contributed the largest sales share.

### Operational Insights
- Product category analysis helps identify high-demand items.
- Outlet analysis assists in optimizing inventory distribution.
- Location-based insights support expansion strategies.
